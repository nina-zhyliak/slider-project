Parameter Default Valid Values Specification
boxWidth 1500 '>= 0' Sets the width of the main container (in 'px').
boxHeight 700 '>= 0' Sets the height of the main container (in 'px').
intervalStart false true/false Self-running slide show
btnPause false true/false pause/play button (used for intervalStart). Recommended to turn off when the main container is small
contrInterval true true/false controlled slide show with the button
speed 1000 '>= 0' slide show speed
slideColumn false true/false slide arrangement in columns (cannot be used with slideTape!). Recommended to use for an even number of slides.
slideTape false true/false Arrange slides in a row (cannot be used with slideColumn!). Recommended to use for an odd number of slides.
borderColorActiveSlide 'yellow' any color Active slide border color
generalSlide true true/false disables slide broadcast
arrows true true/false connect / disconnect arrows for turning. Recommended to turn off when the main container is small
